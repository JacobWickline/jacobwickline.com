Raspberry Pi Beowulf Cluster

Parallel Processing

This project consisted of myself and three other students, our goal was to construct a Beowulf Cluster out of four Raspberry Pis using OpenMPI and mpi4py to run the Raspberry Pis in parallel to process large data files or large amounts of computing. This Beowulf Cluster is being used as a password cracker that utilizes a brute force method to cycle through all possible letter and symbol combinations for passwords. My team and I are running tests comparing a single Raspberry Pi vs. four Raspberry Pis.

Technologies Used:

Python, Open MPI, mpi4py, and Raspberry Pi
