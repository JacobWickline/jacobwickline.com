PyQt5 Painting Application

Draw Something

This project is a simple painting application. The user can increase the pixel size, change the paint color, save the image, and clear the canvas all in the tool bar. The app also records and saves all mouse coordinates as you draw

Technologies Used:

Python and PyQt5
